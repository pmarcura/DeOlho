# Código de Conduta

## Compromisso

Como comunidade, queremos que o DeOlho seja um espaço profissional, inclusivo e responsável para construir tecnologia cívica aberta.

Este código se aplica a issues, pull requests, discussões, documentação, eventos e qualquer espaço associado ao projeto.

## Comportamentos esperados

- Tratar outras pessoas com respeito.
- Discordar com clareza técnica, não com ataque pessoal.
- Explicar decisões e tradeoffs de forma verificável.
- Corrigir erros publicamente quando necessário.
- Respeitar privacidade, segurança e limites legais.
- Evitar linguagem que incentive perseguição, assédio ou exposição indevida.

## Comportamentos inaceitáveis

- Assédio, intimidação, insultos ou ataques pessoais.
- Linguagem discriminatória ou sexualizada.
- Publicação de dados pessoais sem base legal e necessidade clara.
- Acusações de corrupção, crime ou má-fé sem evidência documentada.
- Uso do projeto para perseguição política, partidária ou pessoal.
- Tentativas de burlar moderação, segurança ou revisão.

## Responsabilidade cívica

O DeOlho trabalha com dados públicos sensíveis em contexto político e institucional. Por isso, contribuições devem ser especialmente cuidadosas com:

- distinção entre fato oficial, explicação e opinião;
- sinais de atenção que não indicam irregularidade;
- limitações das fontes públicas;
- proteção de pessoas físicas comuns;
- correção rápida de erros.

## Aplicação

Mantenedores podem remover conteúdo, editar discussões, fechar issues, bloquear usuários ou tomar outras medidas proporcionais quando este código for violado.

Relatos devem ser feitos por canais privados dos mantenedores quando envolverem segurança, assédio ou dados pessoais. Enquanto não houver canal dedicado, abra uma issue apenas se o relato puder ser público sem expor terceiros.

## Atribuição

Este código é inspirado no Contributor Covenant e adaptado para as necessidades de um projeto brasileiro de transparência cívica.
